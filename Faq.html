1. "The mod takes a very long time to start."

  • Do not touch the game for 5 minutes and it will launch. This is a problem with the game engine. We recommend to install the game on NVMe

2. "The game crashes on startup."

  • Check if you have installed the mod correctly. Preferably several times.

3. "When will you release the update?."

  • When we calculate that the number of changes is enough to call it an update.

4. "How versions are numbered?."

  • 0.G.D.M where G is global update, D is default update, M is minor update
