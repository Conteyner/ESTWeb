Installation

1. Via Steam

The easiest way to install is via steam

Just open the mod's page in the Steam Workshop, and click on the subscribe button

×

Next, open the Launcher, and activate the mod

×
×

Enjoy

2. Via Github

If you want to have access to the new updates before anyone else,
you can download the github version of the mod

Method 1:

Step 1:
Open our repository in your browser, click code then download zip

×

Step 2:
After downloading the file open explorer, and go to the path:
Documents\Paradox Interactive\Hearts of Iron IV\mod

Step 3:
Now move the downloaded file to the folder "mod"
Open the transfered file and copy descriptor to the folder "mod"
and rename it to "East-Showdown-main"

×

Step 4:
Open it with any text editor at the very end add the line path=""
(add your path to the mod in quotes).
The path must not be in Cyrillic
Replace "\" with "/"

×

Step 5:
Open HOI 4, Playsets, click "ADD MORE MODS" add the mod and turn it on

×
×
×

Enjoy
